**LEGEND**
gp = games played
pts = Points Leaders
reb = Rebounds
ast = assists 
net_rating = measures the plus-minus statistics for a given player
when the player is in the game relative to the plus-minus
statistics when the player is NOT in the game.
oreb_pct = Offensive rebounds
dreb_pct = Defensive rebounds
usg_pct = Players usage
ts_pct = True shooting percentage
ast_pct = Assist Percentage
